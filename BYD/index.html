Kinex

Kinex is an alternative to the default BYD home screen interface. It focuses on being clean and intuitive, while displaying useful controls and information. Includes features like Live telemetry, 3D vehicle model, vehicle controls, custom navigation bar, multiple layouts, embedded apps, blind spot cameras and many more!

BYD Trip Stats is a feature-complete Android analytics dashboard for BYD DiLink vehicles — built by a BYD Seal owner and running on production hardware. It now uses the BYD SDK and operates as a standalone in-car analytics app without relying on an external companion bridge.
